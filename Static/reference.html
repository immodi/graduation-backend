API Documentation

Root

GET ->

Returns a list of supported languages.

Response (200 OK):

{
  "dateTime": "string",
  "statusCode": number,
  "supportedLanguages": ["string"]
}

Login

POST ->

Authenticates a user and returns a JWT token.

Request Body:

{
  "username": "string",
  "password": "string"
}

Response (200 OK):

{
  "dateTime": "string",
  "statusCode": number,
  "token": "string"
}

Response (400/500 Error):

{
  "dateTime": "string",
  "statusCode": number,
  "errorMessage": "string"
}

Compile

POST -> (Requires Authorization)

Compiles code inside a Docker container.

Authorization Note:

For endpoints that require authorization, the token must be provided in the HTTP header as follows:

Authorization: Bearer {token}

Request Body:

{
  "language": "string",
  "codeToRun": "string"
}

Response (200 OK):

{
  "dateTime": "string",
  "statusCode": number,
  "output": "string"
}

Response (400/500 Error):

{
  "dateTime": "string",
  "statusCode": number,
  "errorMessage": "string"
}
